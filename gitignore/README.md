A collection of standard `.gitignore` templates for common languages and frameworks to help keep your repositories clean.

## üìÅ Available Templates

- **Node.gitignore**: Standard ignores for Node.js projects (node_modules, logs, coverage, etc.).
- **Python.gitignore**: Comprehensive ignores for Python projects (venv, __pycache__, .pyc, etc.).

## üöÄ Usage

1. **Choose a template**: Identify the template that matches your project's technology stack (e.g., `Python.gitignore`).
2. **Copy and Rename**: Copy the file to the root of your project and rename it to `.gitignore`.

### Command Line Example

```bash
# For a Python project
cp templates/gitignore/Python.gitignore ./my-python-project/.gitignore

# For a Node.js project
cp templates/gitignore/Node.gitignore ./my-node-project/.gitignore
```
