# 基础 HTML/CSS/JS 模板

[![License: MIT](https://img.shields.io/badge/License-MIT-brightgreen.svg)](https://opensource.org/licenses/MIT)

一个用于构建静态网页或快速原型的极简启动模板，使用原生 HTML、CSS 和 JavaScript。无构建工具，无框架负担。

[English README](./README.md)

## ✨ 特性

- **HTML5 模版**: 语义化结构，开箱即用。
- **基础 CSS**: 包含清理默认样式和基础样式。
- **原生 JS**: 已链接空的脚本文件，随时可以编写逻辑。
- **零配置**: 打开即可开始编码。

## 🚀 快速开始

1. 将此目录复制到你的项目位置。
2. 在浏览器中打开 `index.html`。
3. 开始编辑！

## 📂 项目结构

```text
web-basic/
├── index.html    # 主 HTML 文档
├── style.css     # 样式表
├── script.js     # JavaScript 逻辑
└── README.md     # 文档
```
