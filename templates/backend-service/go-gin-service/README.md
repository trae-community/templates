# Go Gin Service Template

A starter template for building REST APIs with Go and the Gin framework.

## Setup

1. Download dependencies:
   ```bash
   go mod tidy
   ```

2. Run the server:
   ```bash
   go run main.go
   ```

3. The server will run at `http://localhost:8080`.

## Endpoints

- `GET /ping`: Returns JSON `{"message": "pong"}`
