# Rust Actix Web Starter

A starter template for building web services with Rust and Actix Web.

## Setup

1. Ensure you have Rust installed (rustup).

2. Run the server:
   ```bash
   cargo run
   ```

3. The server will run at `http://127.0.0.1:8080`.

## Endpoints

- `GET /`: Returns "Hello World from Rust Actix!"
