# Nuxt.js 启动模板

极简的 Nuxt 3 启动模板。

## 设置

确保安装依赖：

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## 开发服务器

在 `http://localhost:3000` 启动开发服务器：

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev
```

## 生产环境

构建生产环境应用程序：

```bash
# npm
npm run build

# pnpm
pnpm build

# yarn
yarn build
```

查看 [部署文档](https://nuxt.com/docs/getting-started/deployment) 获取更多信息。
