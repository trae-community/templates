# Tailwind CSS Starter

A simple starter template for using Tailwind CSS with HTML.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the build process in watch mode:
   ```bash
   npm run build
   ```

3. Open `src/index.html` in your browser.

## Structure

- `src/input.css`: Your custom CSS and Tailwind directives.
- `src/index.html`: Your HTML file.
- `dist/output.css`: The compiled CSS file (generated by build).
