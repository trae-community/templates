# Tailwind CSS 启动模板

一个使用 HTML 和 Tailwind CSS 的简单启动模板。

## 设置

1. 安装依赖：
   ```bash
   npm install
   ```

2. 在监听模式下启动构建过程：
   ```bash
   npm run build
   ```

3. 在浏览器中打开 `src/index.html`。

## 结构

- `src/input.css`: 自定义 CSS 和 Tailwind 指令。
- `src/index.html`: HTML 文件。
- `dist/output.css`: 编译后的 CSS 文件（由构建生成）。
