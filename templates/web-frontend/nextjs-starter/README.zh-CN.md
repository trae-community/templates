# Next.js 启动模板

配置了 TypeScript 的现代 Next.js 14 模板。

## 快速开始

1. 安装依赖：
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

2. 运行开发服务器：
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

3. 在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看结果。

## 特性

- Next.js 14 (App Router)
- React 18
- TypeScript
- 模块解析：Bundler
